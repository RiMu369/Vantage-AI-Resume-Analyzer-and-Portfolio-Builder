# Vantage AI: Resume Analyzer and Portfolio Builder
Vantage AI is an AI-powered career assistance web application that helps users analyze resumes, generate portfolio content, and prepare for interviews through an interactive chatbot interface.

The system allows users to upload resumes (PDF/DOCX), simulate analysis, and receive structured feedback on skills, missing keywords, and overall quality. It integrates a Generative AI-powered chatbot (via Hugging Face API) that provides real-time guidance on resume improvement and interview preparation.

The platform is designed with a modern, interactive UI featuring drag-and-drop resume upload, progress indicators, and a conversational interface, making career development more engaging and accessible.
